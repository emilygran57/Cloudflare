# Feedback insight demo

A simple prototype tool was created that aggregates user feedback and highlights key insights onto a dashboard.

# Problem- 
Product Managers often recieve an abundant amount of feedback from multiple platforms. It can often be difficult to distinguish priority and trends within the different feedback. This tool was designed to demonstate an efficant way to aggregate and summarize feedback. 

# Features
- Aggregates mock feedback data 
- Displays the total amount of feedback and gatagorizes it
- Provides feedback that should be prioritized

# Cloudflare Products Used
**Workers** - The ability to run the aggregation logic
